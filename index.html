風格資料庫

管理模式已開啟：可在「縮圖」欄逐列上傳圖片，前端會即時顯示。上傳的縮圖暫存在這台瀏覽器，按「匯出 styles.js」可永久寫入。
縮圖 英文提示詞 中文提示詞 圖片說明
每列分四欄：縮圖｜英文提示詞｜中文提示詞｜圖片說明。點各欄「複製」可複製提示詞。
右下角「管理模式」開啟後，可逐列上傳縮圖、新增風格與匯出。

管理模式

管理模式下，表格「縮圖」欄會出現上傳按鈕，可逐列換上自己生成的縮圖。新增的風格與上傳的縮圖都先存在這台瀏覽器（localStorage），按「匯出 styles.js」可下載永久版本覆蓋原檔。

新增風格

自訂風格：0 個；已上傳縮圖：0 張。
「匯出 styles.js」會下載含內建＋自訂＋已上傳縮圖的完整 styles.js，覆蓋專案同名檔即可永久保存、上傳 GitHub Pages。
